# 📢 AdBot — Telegram-бот для размещения рекламы

Бот для автоматизированного размещения платной рекламы в Telegram-каналах с оплатой через **CryptoBot** (@send) и системой донатов.

---

## ✨ Возможности

- **2 рекламных ячейки** — одновременно может быть активно не более 2 рекламных постов.
- **Оплата через CryptoBot** — выставление счёта в USDT/RUB, автоматическая проверка оплаты.
- **Донаты 🧁** — поддержка проекта с возможностью оставить комментарий админу.
- **Предпросмотр поста** — пользователь видит, как будет выглядеть пост, до оплаты.
- **До 5 фотографий** — поддержка медиагрупп в рекламном посте.
- **Автоматическое удаление** — пост удаляется из канала через 24 часа.
- **Уведомления** — все пользователи получают уведомление, когда ячейка освобождается.
- **Фильтрация контента** — жёсткий запрет на рекламу наркотиков, казино и азартных игр.
- **Панель администратора** — установка цены и времени ожидания оплаты.

## 🚀 Быстрый старт (VPS)

### 1. Клонирование репозитория
```bash
git clone https://github.com/B3B3097/Bott.git
cd Bott
```

### 2. Установка зависимостей
```bash
pip install aiogram aiohttp
```

### 3. Настройка
Откройте `main_vps.py` и заполните переменные в секции КОНФИГУРАЦИЯ:
- `API_TOKEN`
- `CRYPTO_PAY_TOKEN`
- `ADMIN_ID`
- `CHANNEL_ID`

### 4. Запуск
```bash
python3 main_vps.py
```

## 👑 Команды администратора

| Команда | Описание |
|---|---|
| `/admin` | Панель управления (цена, тайм-аут) |
| `/setprice [число]` | Установить цену за 24 часа (в USDT) |
| `/settimeout [число]` | Установить время ожидания оплаты (в минутах) |

---

## 🔄 Процесс работы

1. **Реклама:** Пользователь создает пост -> Оплачивает -> Пост в канале -> Удаление через 24ч -> Рассылка о свободной ячейке.
2. **Донат:** Пользователь вводит сумму -> Оплачивает -> Пишет комментарий -> Админ получает уведомление.
