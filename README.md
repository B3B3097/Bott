# 📢 AdBot — Telegram-бот для размещения рекламы

Бот для автоматизированного размещения платной рекламы в Telegram-каналах с оплатой через **CryptoBot** (@send).

---

## ✨ Возможности

- **2 рекламных ячейки** — одновременно может быть активно не более 2 рекламных постов
- **Оплата через CryptoBot** — выставление счёта в USDT, автоматическая проверка оплаты
- **Предпросмотр поста** — пользователь видит, как будет выглядеть пост, до оплаты
- **До 5 фотографий** — поддержка медиагрупп в рекламном посте
- **Автоматическое удаление** — пост удаляется из канала через 24 часа
- **Уведомления** — все пользователи получают уведомление, когда ячейка освобождается
- **Фильтрация контента** — жёсткий запрет на рекламу наркотиков, казино и азартных игр
- **Панель администратора** — установка цены за 24 часа

## 🚀 Быстрый старт

### 1. Клонирование репозитория

```bash
git clone https://github.com/B3B3097/Bott.git
cd Bott
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Настройка переменных окружения

Скопируйте `.env.example` в `.env` и заполните значения:

```bash
cp .env.example .env
nano .env
```

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен бота от @BotFather |
| `CRYPTO_PAY_TOKEN` | API-токен от @CryptoBot -> My Apps -> Create App |
| `ADMIN_ID` | Ваш Telegram User ID (узнать у @userinfobot) |
| `CHANNEL_ID` | ID канала, куда публикуются посты (например -1001234567890) |
| `DEFAULT_PRICE` | Начальная цена за 24 часа в USDT |

### 4. Добавление бота в канал

Добавьте бота в ваш канал как **администратора** с правами:
- Публикация сообщений
- Удаление сообщений

### 5. Запуск

```bash
python bot.py
```

|---|---|
| `/admin` | Открыть панель администратора |

## 🔄 Процесс размещения рекламы

```
Пользователь нажимает "Купить рекламу"
        ↓
Бот показывает количество свободных мест и цену
        ↓
Пользователь отправляет текст поста
        ↓  (фильтрация запрещённого контента)
Пользователь отправляет до 5 фотографий (или пропускает)
        ↓
Бот показывает предпросмотр поста
        ↓
Пользователь нажимает "Продолжить к оплате"
        ↓
Бот создаёт счёт через CryptoBot
        ↓
Пользователь оплачивает и нажимает "Проверить оплату"
        ↓
Пост публикуется в канале с тегом #РЕКЛАМА
        ↓
Через 24 часа пост удаляется, все пользователи получают уведомление
```

## 🗂 Структура проекта

```
Bott/
├── bot.py              # Точка входа
├── config.py           # Конфигурация
├── database.py         # Работа с SQLite
├── requirements.txt    # Зависимости
├── .env.example        # Шаблон переменных окружения
├── handlers/
│   ├── user.py         # Обработчики пользователей (FSM)
│   └── admin.py        # Обработчики администратора
├── services/
│   └── crypto_pay.py   # Интеграция с CryptoPay API
└── utils/
    ├── filters.py      # Фильтрация запрещённого контента
    └── scheduler.py    # Планировщик проверки ячеек
```
